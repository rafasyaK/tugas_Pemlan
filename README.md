# Simulasi Akun Bank

Simulasi sistem perbankan sederhana menggunakan Java sebagai tugas Pemrograman Lanjut Semester 2.

## Fitur
- Buat akun bank baru
- Cek saldo
- Deposit
- Tarik tunai
- Transfer antar rekening (coming soon)

## Konsep OOP
- Encapsulation (private attribute, getter/setter dengan validasi)
